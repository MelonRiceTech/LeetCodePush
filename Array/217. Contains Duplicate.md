####217. Contains Duplicate

##### Difficulty: Easy

```
class Solution {
    public boolean containsDuplicate(int[] nums) {
        if (nums.length <= 1){
            return false;
        }
        HashSet<Integer> hash = new HashSet<Integer>();
        for (int i = 0;i < nums.length;i++){
            if (!hash.add(nums[i])){
                return true;
            }
        }
        return false;
    }
}
```
